- 완전탐색
	- 정의
		- 모든 경우의 수를 탐색하는 알고리즘
	- 문제 풀이 팁
		- 문제의 최대 범위를 보고 시간 복잡도가 1억 미만이면 일단 해보자
		- 순서 상관 있은 문제는 순열
		- 순서 상관 없으면 조합
	- 종류
		- 반복문을 활용한 방법
			- for, while문을 이용한 로직을 짜는 방법
		- 재귀 함수를 이용한 방법
			- 반복문으로 가능하면 무조건 반복문 사용
			- 너무 복잡하거나, 어떤 행위를 반복하는데 매개변수만 변경하면 될것같으면 재귀함수 사용
			- 조합 or 순열 + DFS/DFS 등 알고리즘
			- 경우의 수 마다 생각해야 되는 로직도 나옴
