- RS(Rasterizer State)
	- 정의
		- 렌터링 파이프라인에서 렌더링 방식에 대한 설정을 정의하는 객체
		- GPU가 객체를 화면에 어떻게 그릴지 제어
		- 레스터라이저 단계에서 3D공간의 기하 데이터를 2D화면에 투영하는 과정

- Sampler State
	- 텍스처 샘플링(샘플링 방식과 필터링 방식 결정)하는 방법에 대한 설정하는 객체

- Blend State
	- 정의
		- 픽셀 세이더 단계에서 출력된 색상 값을 기존 렌더 타깃에 있는 색상 값과 혼합하는 방법을 정의하는객체
		- 투명, 반투명, 그림자등 효과를 구현
